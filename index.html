Dicas

Saudações
Você pode conversar com o agente.

Digite ou fale comandos como: "Bom dia", "Olá", "Oi" ou "Tudo bem?" para que o agente converse com você.

Data e hora
Você pode pergutar a data e a hora atual.

Digite ou fale comandos como: "Que horas são?" ou "Que dia é hoje?" para que o agente informe o dia ou a hora.

Páginas da Web
Você pode abrir as principais redes sociais.

Digite ou fale o comando: "Abra meu ..." e informe a rede social para que o agente abra a respectiva página para você. O agente abre atualmente o Google, Facebook, Instagram e Twitter.

Sobre
Sobre este projeto

Este projeto foi desenvolvido para a discipina de Inteligência Artificial e Computacional no curso de Ciência da Computação do Centro Universitário Anhnaguera de Niterói - UNIAN. Foi consumida neste programa a API Artyom.js onde a mesma faz união de duas outras APIs nativas de alguns navegadores: speechSynsthesis e webkitSpeechRecognition. Desenvolvido por Darlan Caruso, código fonte para consulta disponível neste link.

Skills: HTML, CSS, JavaScript, jQuery, Semantic UI, Babel, Node e PWA.
